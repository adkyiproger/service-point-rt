Редактор шаблонів

Для того щоб під час друку на шаблоні відобразилося значення з заявки вставте в потрібно для вам місці змінну з таблиці.

Ім'я Змінної Опис
ISSUE_ID Номер заявки, зазвичай можна використовувати для заголовку в акті виконаних робіт
FNAME Ім'я клієнта
STREET Адреса клієнта
PHONE Телефон Клієнта
DEVICE_NAME Назва пристрою
DEVICE_NUMBER Номер Пристрою
DEVICE_TYPE Тип пристрою (Телефон, Смартфон,...)
START_DATE Дата прийому
END_DATE Дата виконання
COMMENTS Коментарі до заявки
DESCRIPTION Опис пристрою
WARRANTY_NAME Тип гарантії
WARRANTY_TEXT Текст гарантії
WORKLOG_DETAILS Деталі роботи (список виконаних робіт), вартість
WORKLOG_TOTAL Загальна вартість роботи
DISCOUNT_VALUE Знижка, сума знижки
REPLACEMENT_DETAILS Замінені деталі, список замінених деталей, кількість, вартість
REPLACEMENT_TOTAL Загальна вартість замінених деталей
TOTAL_COST_WITHOUT_DISCOUNT Загальна вартість без знижки
TOTAL_COST_WITH_DISCOUNT Загальна вартість зі знижкою
